# Docker Nginx-Tomcat
## 도커를 이용해서 nginx 와 tomcat 을 연동하는 프로젝트를 진행하였습니다.

공부 목적으로 진행하였고 지속적으로 업데이트 진행할 예정입니다.

필요한 command
----------

- build / up

    실행하기전 빌드시키는 걸 좋아한다.
    ```
    docker-compose up -d --build
    ```
- shutdown
    ```
    docker-compose down
    ```

같이 참고하면 좋을 거 같아요
